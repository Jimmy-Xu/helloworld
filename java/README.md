Run helloworld under macOS with java
---------------------------------------

# run helloworld

```
//check java
$ which java
/usr/bin/java
$ java -version
java version "1.8.0_162"
Java(TM) SE Runtime Environment (build 1.8.0_162-b12)
Java HotSpot(TM) 64-Bit Server VM (build 25.162-b12, mixed mode)

//build
$ javac HelloWorld.java

//run
$ java HelloWorld
Hello,World!
```
